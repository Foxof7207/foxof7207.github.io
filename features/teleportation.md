# Teleportation

Getting around the world quickly.

| Command | Description |
| :--- | :--- |
| `/tpa <player>` | Send a teleport request to another player. |
| `/back` | Teleport back to your previous location. |
| `/warp <name>` | Teleport back to the specified warp. |
| `/suicide` | Instantly kill you. |
