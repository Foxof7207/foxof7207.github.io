# Lifesteal

Volez des cœurs à vos ennemis et devenez le survivant ultime !

### Mécaniques
- **Lifesteal** : Gagnez un cœur lorsque vous tuez un joueur.
- **Élimination** : Si vous perdez tous vos cœurs, vous êtes éliminé jusqu'à ce qu'un coéquipier vous ressuscite.
- **Retrait** : Transformez vos cœurs supplémentaires en objets physiques pour les échanger ou les stocker.
- **Résurrection** : Utilisez un **Cristal de résurrection** pour ramener un coéquipier éliminé.

| Commande | Description |
| :--- | :--- |
| `/hearts` | Vérifiez votre nombre de cœurs actuel. |
| `/withdrawheart <nombre>` | Retirez des cœurs sous forme d'objets. |
| `/revive <joueur>` | Ressuscitez un joueur éliminé. |
| `/lifestealz` | Ouvrez le menu principal Lifesteal. |
