# Téléportation

Se déplacer rapidement à travers le monde.

| Commande | Description |
| :--- | :--- |
| `/tpa <joueur>` | Envoyer une demande de téléportation à un autre joueur. |
| `/back` | Se téléporter à votre emplacement précédent. |
| `/warp <nom>` | Se téléporter au warp spécifié. |
| `/suicide` | Se tuer instantanément. |
